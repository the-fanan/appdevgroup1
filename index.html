Steps to recreate project.

  1. Downloads
    • Download Atom
    • Register on Git hub for an account
    • Install Git
    • Register for a Heroku account
    • Install Heroku CLI
  2. Setup
    • Clone repository into local machine
    • Make updates
  3. Live preview
    • Create composer.json file
    • Add the following lines:

      {
      "name": "appdevgroup1/appdevgroup1",
      "description": "Appdev group 1 project",
      "keywords": ["group work", "school"],
      "license": "GNU",
      "type": "project",
      "require": {
      "php": ">=7.1.3"
      }
      }

    • Push to heroku
